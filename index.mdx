## What is 2048?

2048 is a classic single-player puzzle game played on a 4×4 grid. You slide numbered tiles in four directions — left, right, up, or down. When two tiles with the same number collide, they merge into one tile with their combined value. The goal is to create a tile with the number **2048**.

This project offers two ways to play:

<CardGroup cols={2}>
  <Card title="Desktop App" icon="desktop" href="/guides/desktop-app">
    A native Python GUI built with Tkinter. Runs locally with keyboard controls (Arrow keys or WASD). No browser needed.
  </Card>
  <Card title="Web App" icon="globe" href="/guides/web-app">
    A Flask-powered web server with a JavaScript frontend. Play in any browser on your local machine.
  </Card>
</CardGroup>

## Key features

<CardGroup cols={2}>
  <Card title="Score tracking" icon="trophy">
    Tracks your current score and best (high score) across every session. Scores update in real-time as you merge tiles.
  </Card>
  <Card title="Keyboard controls" icon="keyboard">
    Full support for Arrow keys and WASD in the desktop app. The web version responds to Arrow keys via JavaScript.
  </Card>
  <Card title="Dynamic tile colors" icon="palette">
    Every tile value from 2 to 2048 has a distinct color, making it easy to spot merge opportunities at a glance.
  </Card>
  <Card title="Game over detection" icon="circle-xmark">
    The game automatically detects when no more moves are possible and presents a replay prompt with your final score.
  </Card>
</CardGroup>

## Quick install

```bash
# Clone the repository
git clone https://github.com/NiwinKumar/2048-TheGame.git
cd 2048-TheGame

# Install dependencies
pip install numpy flask

# Run the desktop version
python 2048.py

# Or run the web version
python app.py
```

<Note>
  Python 3.x is required. The desktop app uses `tkinter` (bundled with Python on most platforms) and `numpy`. The web app additionally requires `flask`.
</Note>
