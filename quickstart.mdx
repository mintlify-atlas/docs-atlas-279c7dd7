## Prerequisites

Before you begin, make sure you have the following installed:

- **Python 3.x** ‚ÄĒ download from [python.org](https://www.python.org/downloads/) if not already installed.
- **pip** ‚ÄĒ bundled with Python 3.4 and later. Verify with `pip --version`.

<Note>
  The desktop app requires `tkinter`, which is included with the standard Python distribution on Windows and macOS. On Debian/Ubuntu Linux, install it separately: `sudo apt-get install python3-tk`.
</Note>

## Installation and setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/NiwinKumar/2048-TheGame.git
    ```
  </Step>

  <Step title="Enter the project directory">
    ```bash
    cd 2048-TheGame
    ```
  </Step>

  <Step title="Install dependencies">
    Both the desktop app and the web app share the same dependency list.

    ```bash
    pip install numpy flask
    ```

    - `numpy` ‚ÄĒ provides the `np.zeros` 4√ó4 integer matrix used as the game board.
    - `flask` ‚ÄĒ powers the web server and JSON API (not needed if you only plan to run the desktop version).
  </Step>

  <Step title="Run the desktop app or the web app">
    Choose the version you want to play.

    **Desktop app (Tkinter)**

    ```bash
    python 2048.py
    ```

    A native window opens immediately. Use the arrow keys or WASD to slide tiles.

    **Web app (Flask)**

    ```bash
    python app.py
    ```

    Then open your browser and navigate to:

    ```
    http://localhost:5000
    ```

    The game loads in the browser. Use the arrow keys to play.
  </Step>
</Steps>

## What you'll see

### Desktop app

When `2048.py` starts, a 4√ó4 grid appears with two tiles already placed (each showing either **2** or **4**). The window title is `2048` and the background matches the original game's `#faf8ef` cream tone. Your current score and high score are displayed above the grid and update in real time as you merge tiles.

Tile colors follow the standard 2048 palette ‚ÄĒ light beige for low values, shifting through orange and red toward golden yellow as values increase toward 2048:

```python
colors = {
    2: '#eee4da', 4: '#ede0c8', 8: '#f2b179', 16: '#f59563',
    32: '#f67c5f', 64: '#f65e3b', 128: '#edcf72', 256: '#edcc61',
    512: '#edc850', 1024: '#edc53f', 2048: '#edc22e'
}
```

### Web app

When `app.py` starts, Flask launches a development server in debug mode:

```python
if __name__ == '__main__':
    app.run(debug=True)
```

Opening `http://localhost:5000` loads the HTML template served by the root route. The board renders identically to the desktop version, with the same tile colors and score display. Every keypress triggers a POST to `/move`, and the board re-renders on the response.

<Tip>
  The Flask development server is single-threaded and not suitable for production. For personal or local use it works perfectly ‚ÄĒ no extra configuration needed.
</Tip>
